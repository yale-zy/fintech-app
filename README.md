# WealthPro

A React-based investment app supporting funds, stocks, and wealth management products with simulated trading.

## Requirements

- Node.js 22 LTS — download at: https://nodejs.org

## Getting Started

```bash
# 1. Enter the project directory
cd fintech-app

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

Then open http://localhost:5173 in your browser.
